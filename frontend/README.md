# PrimeTrade Frontend (React + Vite)

Dark theme notes app with auth.

## Quick Start

```bash
npm install
npm run dev
```

http://localhost:5173

## Backend API

Backend runs on http://localhost:8080/api

See root README.md for full setup.

## Features

- Dark grey theme (Sass)
- Auth (login/register/protected)
- Notes CRUD
- Responsive
